# Como acessar o sistema

## Primeiro acesso

Para realizar o primeiro acesso na nossa plataforma, acesse https://ipontoweb.com.br e clique em `Registrar`.

![Tela de login](images/tela-login.png)

Na tela de cadastro, é necessário que seja preenchido os seguintes dados:
1. Chave de acesso: Essa chave é disponibilizada por um de nossos revendedores autorizados. Em caso de dúvidas, contate o revendedor.
2. CPF/CNPJ: CPF ou CNPJ associado a chave de acesso
3. Email: Email associado a chave de acesso
4. Senha: Senha de acesso a plataforma

![Tela de registro](images/tela-registro.png)

> Em caso de problemas ao realizar o registro, contate o revendedor autorizado

---

## Acesso à plataforma

Para realizar o acesso à nossa plataforma, acesse https://ipontoweb.com.br e preencha os dados cadastrados.
